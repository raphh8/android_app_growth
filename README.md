# Growth

Growth est une application Android permettant d'organiser ses séances de sport.

## Fonctionnalités

* Outil de recherche : grâce à l'API ExerciseDB, l'utilisateur peut retrouver plus de 1500 exercices de musculation, avec un GIF montrant la réalisation du mouvement et d'autres informations comme les muscles ciblés ou les équipements nécessaires.
* Création de séance : l'utilisateur peut créer ses propres séances en choisissant les exercices. Il peut indiquer le nombre de série, répétition et poids pour personnaliser son entrainement. Une estimation de temps pour la réaliser est calculée automatiquement.
* Semainier : l'utilisateur peut organiser ses séances dans un semainier. Chaque séance programmée sera notifiée la veille au soir.
* Chronomètre : un chronomètre est inclus dans l'application, utilisable entre les séries pour prendre le bon temps de repos sans quitter l'application et continuer à suivre sa séance.
