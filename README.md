# Growth

Growth est une application Android permettant d'organiser ses séances de sport.

## Fonctionnalités

* Outil de recherche : grâce à l'API ExerciseDB, l'utilisateur peut retrouver plus de 1500 exercices de musculation, avec un GIF montrant la réalisation du mouvement et d'autres informations comme les muscles ciblés ou les équipements nécessaires.
* Création de séance : l'utilisateur peut créer ses propres séances en choisissant les exercices. Il peut indiquer s'il s'agit d'étirements, de cardio ou d'exercices de musculation classiques, puis les paramétrer en fonction (nombre de série, répétitions, poids, durée, distance...) pour personnaliser son entrainement. Une estimation de temps total pour réaliser la séance est calculée automatiquement.
* Semainier : l'utilisateur peut organiser ses séances dans un semainier. Chaque séance programmée sera notifiée la veille au soir.
* Minuteur : un minuteur est inclus dans l'application, utilisable entre les séries pour prendre le bon temps de repos sans quitter l'application et continuer à suivre sa séance.
